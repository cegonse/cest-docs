— Cest is a C++ testing framework focused on simplicity and readability —

Main Features

Single Header
All features are included in a single C++ header,
making it easy to integrate with Make or CMake

BDD inspired API
Using from it to expect, write your tests
as you would in JavaScript or Ruby.

JUnit Export
Generate JUnit compatible XML reports
to include in your CI pipeline.
